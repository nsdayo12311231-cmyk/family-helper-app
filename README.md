- **フロントエンド**: React 18 + TypeScript
- **スタイリング**: Tailwind CSS 3.4
- **ビルドツール**: Vite 7.1
- **ルーティング**: React Router DOM
- **アイコン**: Lucide React
- **データ管理**: LocalStorage + React Context
